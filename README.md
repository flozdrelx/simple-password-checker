# Simple Password Checker

#### For the portfolio

A simple script that will check your password, give you a score, and based on that, give you suggestions and also perform a password entropy analysis.

---

# How to use it

## If you are on Windows:
* Make sure you have python installed
* Double click run.bat
* The script will activate the venv and install the required packages

## If you are on Linux/MacOS:
* Make sure you have python installed (mostly in MacOS)
* Open the terminal in the PasswordChecker folder
* Make it executable:
```bash
chmod +x run.sh
```
* Run the script:
```bash
./run.sh
```
